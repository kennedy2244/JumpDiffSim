# Theoretical Moments of the Log-Return Distribution

Theoretical Moments of the Log-Return Distribution

## Usage

``` r
jumpMoments(object)
```

## Arguments

- object:

  A
  [JumpDiffModel](https://kennedy2244.github.io/JumpDiffSim/reference/JumpDiffModel-class.md)
  object.

## Value

Named numeric vector: mean, variance, skewness, kurtosis.
