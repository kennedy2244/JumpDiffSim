# Compute Negative Log-Likelihood

Compute Negative Log-Likelihood

## Usage

``` r
loglik(object, log_returns, ...)
```

## Arguments

- object:

  A
  [JumpDiffModel](https://kennedy2244.github.io/JumpDiffSim/reference/JumpDiffModel-class.md)
  object.

- log_returns:

  Numeric vector of log asset returns.

- ...:

  Passed to methods.

## Value

Scalar negative log-likelihood value.
